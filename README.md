# Dhreeti Kumar Sahoo - Portfolio

A responsive React portfolio for Dhreeti Kumar Sahoo, an MCA graduate focused on Java full stack development, Spring Boot APIs, Django applications, databases, authentication, and practical web integrations.

Live portfolio: [Dhreeti-Kumar-Sahoo](https://dhreetigithub3.github.io/Dhreeti-Kumar-Sahoo/)

## Overview

This portfolio presents a developer profile, technical skills, resume details, featured projects, and contact channels in one recruiter-friendly web experience. It is built with React and Vite, with a clean component structure and interactive project case studies.

## Highlights

- Responsive single-page portfolio built with React.
- Smooth navigation across hero, about, skills, resume, projects, and contact sections.
- Interactive project filtering for Java Spring Boot and Python Django work.
- Project detail modal with architecture, features, challenges, and implemented approach.
- Resume download support through the public resume asset.
- Contact form that opens a prefilled email using `mailto:`.
- Professional profile links for GitHub and LinkedIn.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

The app will start on the local Vite development server shown in the terminal.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Lint

```bash
npm run lint
```
